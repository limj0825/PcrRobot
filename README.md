# PcrRobot

公主连结QQBot

作者：Leeeeeeee

supported by [lua-miria](https://github.com/only52607/lua-mirai)

## 部署
---
1. 安装好java
  
```
  yum -y install java-1.8.0-openjdk
```

2. 下载最新的[release](https://github.com/only52607/lua-mirai)放在当前的目录下



3. 启动
   
```
  java -jar lua-mirai.jar exec main.lua
```
